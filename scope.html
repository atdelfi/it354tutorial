Scope

Scope is what the function or the program can see. In the example I have var name set as Tyler outside of the function and call set it as Anthony in the function so that it returns Anthony and not Tyler. But if I were not to set it in the function it would return Tyler. So the Function can look out of itself but nothing but it can look inside it.
