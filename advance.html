Advance Arrays


Filter adds what matches the parameter to an new array.


Find, finds the first matching parameter and returns that.


FindIndex, finds the first matching parameter and returns that numbers array spot and if it doesnt find 1 it returns -1.


ForEach, returns each value in an array as a single value.


Map, allows you to alter an array and put it into a new one.


Pop, takes the last array value and returns it, and drops it from the array


Push, allows you to add to your array


Sort, puts the array into accending order.
