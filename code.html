Value

There are five type values in JavaScript - String, Number (Int, Double, Float), Boolean, Null, and Undefined

Unary Operator

There are three main Unary operators, Plus(+), Negetive(-), and Not(!).
Plus(+) changes String, boolean and Null into number values
Negetive(-) changes String, boolean and Null into number values just like Plus but with negation

Hoisting

Hoisting is where the compiler takes all named variables and named functions before compiling
