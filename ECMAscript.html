ECMAScript

ECMAscript is a scripting-language that was standardized by Ecma International. It was created to help standardize JavaScript. JS has be the most widely used implementation of ECMAscript.

History

The creator of JavaScript is Brendan Eich. He started as the chief technology officer at Mozilla. He explains to the interviewer(Paul Krill) that Javascript started as Mocha then LiveScript, then Netscape and finally Javascript all within a six month period. They also ask if ECMAscript and JavaScript are one in the same. Eich stating that it ECMAscript is the standardize name as multiple companies couldn't pool enough money to get a trademark.
